# Б21 — Gollwitzer 1999 — implementation intentions

Классическая статья о **implementation intentions** — заранее заданных **если–то** планах, которые связывают **ситуацию** с **конкретным** действием и повышают вероятность его выполнения по сравнению с одними только намерениями.

## Библиография

- Gollwitzer, P. M. (1999). Implementation intentions: Strong effects of simple plans. *American Psychologist*, *54*(7), 493–503. https://doi.org/10.1037/0003-066X.54.7.493
