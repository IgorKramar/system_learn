# Б34 — Roediger Karpicke 2006 — test enhanced learning

Классическая демонстрация **testing effect** на **прозаическом** тексте: при **кратком** интервале финального теста «побеждает» большее число **чтений**, при **задержке** (например, неделя) — большее число **тестов** извлечения. Условия **SSSS / SSST / STTT** (четыре слота обучения: S = study, T = test) показывают **перекрёст** кривых по времени удержания.

## Библиография

- Roediger, H. L., III, & Karpicke, J. D. (2006). Test-enhanced learning: Taking memory tests improves long-term retention. *Psychological Science*, *17*(3), 249–255. https://doi.org/10.1111/j.1467-9280.2006.01693.x
